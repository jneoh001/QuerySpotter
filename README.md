# About

PostgreSQL query visualizer using Flask.

# Usage

1. Install required packages
`pip install -r requirements.txt`

2. Activate virtual environment

- Windows `env/Scripts/activate`
- MacOS/Linux `source env/bin/activate`

3. Run Server

`cd backend`
`python backend.py`
